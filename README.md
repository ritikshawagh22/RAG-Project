# Educational RAG Tutor

## Description

Educational RAG Tutor is a Python application that allows users to upload a PDF and ask questions based on its content. It retrieves relevant information from the document and generates accurate answers.

## Features

* Upload PDF files
* Ask questions from the document
* Retrieve relevant information
* Simple GUI interface

## Technologies Used

* Python
* Tkinter
* PyPDF
* NumPy
* OpenAI API

## Run

```bash
python main.py
```

## Purpose

To demonstrate a basic Retrieval-Augmented Generation (RAG) system for educational documents.
